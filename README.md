# 3차원 복원 및 객체 검출 기반 건물 균열 탐지

3차원 복원 기법을 통해 건물의 전체 형상을 복원하며 동시에 객체 검출 기법을 활용하여 건물 균열 위치를 탐지

------

#### **2022 소프트웨어공학종합프로젝트 T6 Harbor**

- 팀장 : 소프트웨어공학과 김혜근
- 팀원 : 소프트웨어공학과 양석영
- 팀원 : 소프트웨어공학과 이승준

------

##### **개발 환경**

- Ubuntu 20.04

- Docker 20.10.14

- Docker-compose 1.28.5
